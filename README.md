Octopress-Slideshare-Plugin
===========================

Helps Embed a Slideshare Presentation in Octopress.

Usage
=====

- Install ``` slideshare.rb ``` in your ``` plugins ``` folder.
- Place ``` {% slideshare ID %} ``` in your MD file
